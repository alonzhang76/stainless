# 泰坦福不锈钢管理系统 - 新手操作流程指南

## 一、系统概述

欢迎使用泰坦福不锈钢管理系统！这是一个专为不锈钢贸易企业设计的综合管理平台，涵盖采购、销售、库存、财务、发票等核心业务流程。

### 系统功能模块

| 模块 | 功能说明 |
|------|----------|
| 首页 | 订单流程跟踪、任务提醒、最近活动 |
| 采购明细 | 采购订单管理、收退货管理、理算计算 |
| 销售明细 | 销售订单管理、发退货管理、报价管理 |
| 库存管理 | 库存查询、仓库管理、库存同步 |
| 发票管理 | 进项发票、销项发票、发票统计 |
| 收付款管理 | 收款、付款、对账、收支分析 |
| 通讯录 | 客户/供应商管理、收藏功能 |
| 材料知识 | 牌号对照、行业应用、中英文词汇、HS编码 |
| 理算参数 | 不锈钢重量计算公式和参数 |
| 统计与分析 | 销售/采购统计、排名分析、图表展示 |
| 日历记事 | 日程安排、订单日期自动提醒 |
| 备忘录 | 记录重要事项、Kindle阅读风格 |

## 二、快速上手：5分钟完成一个完整业务流程

下面用一个实际案例演示如何完成从采购到收款的完整流程：

### 案例背景

您公司向「上海不锈钢有限公司」采购一批 304 不锈钢冷轧板，然后销售给「杭州机械设备有限公司」。

### 步骤1：创建采购订单

1. 点击左侧菜单「采购明细」
2. 点击「新建采购订单」按钮
3. 填写以下信息：
   - 订单编号：PO-20260709-001
   - 供应商：上海不锈钢有限公司
   - 产品：不锈钢冷轧板
   - 材质：304
   - 规格：1.5mm*1220mm*2440mm
   - 数量：100 张
   - 重量：3540 公斤
   - 单价：18.5 元/公斤
   - 预计到货日期：2026-07-15
4. 点击「保存」

### 步骤2：查看库存同步

1. 点击左侧菜单「库存管理」
2. 您会看到系统已自动从采购订单同步了库存记录
3. 库存数量显示为 3540 公斤

### 步骤3：创建销售订单

1. 点击左侧菜单「销售明细」
2. 点击「新建销售订单」按钮
3. 填写以下信息：
   - 订单编号：SO-20260709-001
   - 客户：杭州机械设备有限公司
   - 产品：不锈钢冷轧板
   - 材质：304
   - 规格：1.5mm*1220mm*2440mm
   - 数量：50 张
   - 重量：1770 公斤
   - 损/溢重量：0
   - 单价：22.0 元/公斤
   - 预计发货日期：2026-07-18
4. 点击「保存」
5. 返回库存管理页面，库存已自动减少到 1770 公斤

### 步骤4：创建发票

1. 点击左侧菜单「发票管理」
2. 点击「新建发票」按钮
3. 创建销项发票：
   - 发票号码：FP-20260709-001
   - 类型：销项发票
   - 客户：杭州机械设备有限公司
   - 订单编号：SO-20260709-001
   - 金额：38940.00 元
   - 税率：13%
   - 税额：5062.20 元
   - 开票日期：2026-07-18
4. 点击「保存」

### 步骤5：创建收款记录

1. 点击左侧菜单「收付款管理」
2. 点击「新建」按钮
3. 创建收款记录：
   - 类型：收款
   - 类别：销售回款
   - 交易对象：杭州机械设备有限公司
   - 金额：44002.20 元
   - 日期：2026-07-20
   - 订单编号：SO-20260709-001
   - 发票号码：FP-20260709-001
   - 支付方式：电汇（默认）
   - 经手人：周瑾（默认）
4. 点击「保存」

### 步骤6：查看订单流程

1. 点击左侧菜单「首页」
2. 在订单流程跟踪表中查看 SO-20260709-001 的完整状态
3. 进度已更新，显示收款完毕

恭喜！您已完成第一个完整业务流程！

## 四、核心业务流程

### 4.1 完整业务流程概览

```
询价 → 报价 → 采购订单 → 入库 → 库存 → 销售订单 → 发货 → 发票 → 收付款
```

### 4.2 流程详细说明

#### 第一步：创建询价单（可选）

1. 进入「采购明细」页面
2. 点击「新建询价单」按钮
3. 填写供应商、产品、材质、规格等信息
4. 点击「保存」完成询价单创建

#### 第二步：创建报价单（可选）

1. 进入「销售明细」页面
2. 点击「新建报价单」按钮
3. 填写客户、产品、单价等信息
4. 点击「保存」完成报价单创建

#### 第三步：创建采购订单

1. 进入「采购明细」页面
2. 点击「新建采购订单」按钮
3. 填写以下关键信息：
   - **订单编号**：唯一标识，建议使用日期+序号格式（如 PO-20260709-001）
   - **供应商**：从通讯录选择或手动输入
   - **产品**：选择产品类型（不锈钢冷轧板、不锈钢焊管等）
   - **材质**：如 304、316L、2205、718 等
   - **规格**：如 1.5mm*1220mm*2440mm
   - **数量/重量**：输入采购数量和重量
   - **单价/总价**：自动计算或手动输入
   - **预计到货日期**：选择日期
4. 点击「保存」完成采购订单创建
5. **注意**：采购订单状态会自动同步到库存、订单流程和日历

#### 第四步：库存管理

1. 进入「库存管理」页面
2. 系统会自动从采购订单同步库存记录
3. 可以手动编辑库存信息，包括：
   - 仓库位置
   - 库存状态（在库、已售、待入库）
   - 备注信息
4. 销售订单创建后，库存数量会自动减少

#### 第五步：创建销售订单

1. 进入「销售明细」页面
2. 点击「新建销售订单」按钮
3. 填写以下关键信息：
   - **订单编号**：唯一标识
   - **客户**：从通讯录选择或手动输入
   - **产品**：系统会自动关联库存中的产品
   - **损/溢重量**：实际重量与理论重量的差异（正数为溢重，负数为损重）
   - **单价/总价**：系统根据调整后的重量自动计算总价
4. 点击「保存」完成销售订单创建
5. **注意**：销售订单创建后会自动扣减库存并同步到日历

#### 第六步：发票管理

1. 进入「发票管理」页面
2. 根据订单类型创建发票：
   - **进项发票**：采购订单对应的供应商发票
   - **销项发票**：销售订单对应的客户发票
3. 填写发票信息：
   - 发票号码
   - 金额、税率、税额
   - 开票日期、到期日期
4. 点击「保存」完成发票创建

#### 第七步：收付款管理

1. 进入「收付款管理」页面
2. 创建收款或付款记录：
   - **收款**：销售回款，关联销售订单和发票
   - **付款**：采购付款，关联采购订单和发票
3. 填写以下信息：
   - **金额**：收付款金额
   - **支付方式**：承兑汇票、电汇、现金、微信、支付宝（默认电汇）
   - **经手人**：默认「周瑾」
   - **关联订单/发票**：选择对应的订单或发票
4. 点击「保存」完成收付款记录

## 五、常用操作指南

### 5.1 数据搜索与筛选

所有表格页面都支持搜索和筛选功能：

1. **搜索框**：输入关键词搜索订单编号、产品、客户、供应商等
2. **状态筛选**：按订单状态、发票状态、收付款状态筛选
3. **日期筛选**：按日期范围筛选记录
4. **重置按钮**：清除所有筛选条件

### 5.2 数据导出

支持将数据导出为 Excel 文件：

1. 在表格页面找到「导出」按钮
2. 点击后自动下载 Excel 文件
3. 支持按当前筛选条件导出

### 5.3 数据备份与恢复

系统提供数据备份和恢复功能，确保数据安全：

1. 点击页面右上角的「数据管理」按钮
2. 选择「备份数据」：系统自动下载备份文件（JSON格式）
3. 选择「恢复数据」：选择之前备份的 JSON 文件进行恢复
4. **注意**：恢复数据会覆盖现有数据，请谨慎操作

### 5.4 清空数据（管理员权限）

仅管理员账号可使用此功能：

1. 点击页面右上角的「数据管理」按钮
2. 选择「清空数据」
3. 输入管理员密码（请联系管理员获取）
4. 点击「确认清空」
5. **注意**：此操作将清空所有业务数据，但保留材料知识和理算参数

## 六、特殊功能说明

### 6.1 损/溢重量处理

销售订单中的「损/溢重量」字段用于处理实际重量与理论重量的差异：

- **正数**：实际重量大于理论重量（溢重），系统自动增加总金额
- **负数**：实际重量小于理论重量（损重），系统自动减少总金额
- **计算公式**：调整后重量 = 原始重量 + 损/溢重量

### 6.2 库存自动同步

系统会自动从采购订单同步库存：

1. 采购订单创建后，系统自动生成库存记录
2. 销售订单创建后，库存数量自动减少
3. 库存完全售出后，记录自动从库存表移除

### 6.3 订单日历自动同步

系统会自动将采购订单和销售订单同步到日历：

1. **采购订单**：自动创建日历事件，显示预计到货日期
2. **销售订单**：自动创建日历事件，显示预计发货日期
3. **优先级自动计算**：
   - 逾期或3天内到期：高优先级（红色）
   - 7天内到期：中优先级（橙色）
   - 其他：低优先级（蓝色）
4. **状态同步**：订单状态变更时，日历事件自动更新

### 6.4 公司名称过滤

系统使用以下两家公司名称进行过滤：
- 无锡泰益特钢有限公司
- 无锡泰坦福新材料有限公司

销售发票通过「销售方」字段匹配公司名称，采购发票通过「购买方」字段匹配。

### 6.5 材料知识查询

在「材料知识」页面可以查询：

1. **牌号对照**：不同标准的不锈钢牌号对应关系（如 GB、ASTM、JIS、DIN 等）
2. **行业应用**：各行业适用的不锈钢材料
3. **中英文词汇**：不锈钢行业专业词汇对照
4. **HS编码**：不锈钢产品的海关编码

### 6.6 理算参数计算

在「理算参数」页面可以查询各类不锈钢产品的重量计算公式：

- 板材：重量 = 厚度 × 宽度 × 长度 × 密度 / 1000000
- 管材：重量 = (外径 - 壁厚) × 壁厚 × 长度 × 密度 × 0.02491
- 棒材：重量 = 直径 × 直径 × 长度 × 密度 × 0.00623
- 密度：不锈钢标准密度为 7.93 g/cm³

### 6.7 备忘录功能

备忘录页面具有以下特点：

1. **Kindle阅读风格**：翻书式阅读体验
2. **双击编辑**：双击正文直接编辑内容，双击标题修改标题
3. **置顶功能**：重要备忘录可以置顶显示
4. **状态管理**：待处理、进行中、已完成三种状态
5. **优先级标记**：高、中、低三种优先级

## 八、常见问题

### Q1：如何创建一个完整的业务流程？

A：按照以下顺序操作：
1. 创建采购订单（采购明细页面）
2. 查看库存同步（库存管理页面）
3. 创建销售订单（销售明细页面）
4. 创建发票（发票管理页面）
5. 创建收付款记录（收付款管理页面）

### Q2：销售订单中的「损/溢重量」是什么？

A：用于处理实际交货重量与订单重量的差异。例如：
- 订单重量 1000kg，实际到货 1050kg，损/溢重量填 +50
- 订单重量 1000kg，实际到货 980kg，损/溢重量填 -20

### Q3：库存数量为什么会自动变化？

A：系统会根据采购订单和销售订单自动更新库存：
- 采购订单创建 → 库存增加
- 销售订单创建 → 库存减少

### Q4：如何关联订单和发票？

A：在创建发票时，填写对应的「订单编号」字段，系统会自动关联。

### Q5：数据如何备份？

A：点击页面右上角的「数据管理」→「备份数据」，系统会自动下载 JSON 格式的备份文件。

### Q6：订单会自动同步到日历吗？

A：是的，系统会自动将采购订单和销售订单同步到日历页面，根据预计到货/发货日期创建事件，并根据订单状态自动计算优先级。

### Q7：如何使用备忘录？

A：在备忘录页面可以：
- 点击「新建」创建新备忘录
- 双击正文直接编辑内容
- 双击标题修改标题
- 点击置顶按钮将重要备忘录置顶
- 拖动侧边栏调整宽度

## 九、注意事项

1. **数据安全**：定期备份数据，以防数据丢失
2. **权限管理**：不同用户有不同权限，请使用正确的账号登录
3. **日期格式**：系统使用 YYYY-MM-DD 格式显示日期
4. **数值精度**：所有金额和重量显示保留两位小数
5. **订单编号**：建议使用唯一的订单编号，便于后续查询和管理
6. **手动编辑**：手动编辑的销售订单（标记为 isManual）不会被自动同步覆盖
7. **日历同步**：订单创建后会自动同步到日历，删除订单时日历事件也会同步删除
